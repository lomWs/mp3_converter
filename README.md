<!-- GETTING STARTED -->
## Getting Started


### Prerequisites

  * Python version 3.10 or higher

### Installation

1. Download "mp3_converter.zip"

2. Pip install
   ```sh
     pip install selenium
     pip install customtkinder
     pip install webdriver-manager
   ```
